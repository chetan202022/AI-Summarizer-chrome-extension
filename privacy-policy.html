Privacy Policy

Last updated: June 2026

Overview

AI Summarizer is a Chrome Extension that generates AI-powered summaries of webpage content using the Groq API. We respect user privacy and do not collect personal data.

Data Collection

This extension does not collect, store, or sell any personal data.

API Key Usage

Users provide their own Groq API key. The API key is:

  • Stored locally using Chrome's chrome.storage.sync
  • Never transmitted to any server other than Groq API
  • Never shared with third parties

External API Usage

The extension sends webpage text to the Groq API to generate summaries. This data is used only for generating responses and is not stored.

Permissions Used

  • activeTab – To read content of current webpage
  • storage – To save user settings (API key, preferences)
  • scripting – To extract page content

Data Security

All sensitive data (API keys and settings) is stored locally on the user's device using Chrome Storage API.

Third-Party Services

  • Groq API (for AI summarization)
