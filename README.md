# AI Summarizer Chrome Extension 🚀

## 📌 Project Overview

AI Summarizer is a Chrome Extension that helps users quickly generate summaries of web content using AI.
It improves productivity by reducing reading time and providing concise information instantly.

## ✨ Features

- 📄 Summarize any webpage content
- ⚡ Fast AI-generated summaries
- 🎯 Clean and simple UI
- 🔌 Easy to use Chrome extension
- 📋 Copy summary with one click

## 🛠️ Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extension APIs
- AI / NLP API

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/chetan202022/AI-Summarizer-chrome-extension.git
```

2. Open Chrome → Go to
```bash
chrome://extensions/
```
3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the project folder

## 🚀 Usage

1. Open any webpage
2. Click on the extension icon
3. Click **Summarize**
4. View AI generated summary
